### 💳 [신용카드 승인 분석](https://github.com/JinMokCho/Credit-Card-Approval-Analysis)
- **설명**: 신용카드 승인 과정을 최적화하기 위해 신청자 데이터를 분석하여 승인율에 영향을 미치는 요인들을 파악하는 프로젝트입니다. 본 연구는 교내 빅데이터 분석 대회에서 우수상을 수상하였습니다.
- **기간**: 2024.11 - 2025.02
- **사용 기술**: Python, SHAP, AutoGluon, Optuna
- **내가 한 성과**: 
    - Kaggle에서 공개된 신용카드 승인 예측 데이터셋을 수집하고, 25,128건의 신청 데이터를 분석하였습니다.
    - 데이터 전처리 과정에서 클래스 불균형 문제를 해결하기 위해 SMOTE 기법을 적용하였습니다.
    - 새로운 변수 생성: 나쁜 빚과 좋은 빚의 비율(Bad_Debt_to_Good_Debt_Ratio), 고객 나이와 근무 연수 비율(Age_to_Working_Years_Ratio), 가족 구성원당 평균 소득(Income_Per_Family_Member) 등을 도출하였습니다.
    - 여러 머신러닝 알고리즘을 사용하여 기준 모델을 구축하고 성능을 평가하였습니다. 모델의 정확도는 95%에 달했습니다.
    - 하이퍼파라미터 튜닝을 통해 모델 성능을 개선하고, StratifiedKFold 교차 검증을 통해 신뢰성을 높였습니다.
    - 주요 변수로는 총 나쁜 빚(Total_Bad_Debt), 총 좋은 빚(Total_Good_Debt), 나쁜 빚과 좋은 빚의 비율(Bad_Debt_to_Good_Debt_Ratio) 및 신청자 나이(Applicant_Age)를 도출하여 신용 카드 승인 결정에 중요한 영향을 미치는 요소로 확인하였습니다.
    - 최종 결과를 바탕으로 은행이 신용카드 승인 결정을 내릴 때 보다 정보에 입각한 판단을 할 수 있도록 기여하였습니다.
